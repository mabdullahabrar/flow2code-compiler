# 📖 Overview

Flow2Code Compiler is an AI-powered visual programming platform that converts flowcharts and natural language prompts into executable source code.

The project integrates:

* Frontend Development
* Backend APIs
* Compiler Design
* Artificial Intelligence
* Machine Learning

to create a beginner-friendly coding environment.

---

# ✨ Features

* 🎨 Interactive flowchart editor
* 🤖 AI-assisted flowchart generation
* 🐍 Python code generation
* ⚙️ C code generation
* 🚀 C++ code generation
* 🧠 Machine learning-based text classification
* 🔄 Compiler engine integration
* 🌐 Flask REST APIs
* 📊 Intelligent pattern recognition

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/mabdullahabrar/flow2code-compiler.git
```

---

## Move Into Directory

```bash
cd flow2code-compiler
```

---

## Install Backend Dependencies

```bash
pip install -r requirements.txt
```

---

## Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## Run React Frontend

```bash
npm start
```

---

## Run Flask Backend

```bash
python app.py
```
